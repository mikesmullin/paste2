# paste2

paste2 is a small static browser tool for converting pasted rich text or HTML into portable markup formats.

It is meant for quick copy/paste cleanup when content needs to move between editors, tickets, chat, and docs without manually rewriting formatting.

Current modes:

- Markdown
- Jira Markup
- Slack mrkdwn
- Markdown to rich text

Live site: https://mikesmullin.github.io/paste2/
